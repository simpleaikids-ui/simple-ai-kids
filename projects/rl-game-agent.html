RL Game Agent

Train an agent to beat a simple game using Stable-Baselines3 + Gymnasium. Watch it go from random flailing to competent play in ~10 minutes.

What You'll Make

A PPO agent trained to balance CartPole, land LunarLander, or play Atari Breakout. You'll record video of before/after and analyze the learning curve.

What You'll Need

  • Google Colab
  • Patience for training runs
  • A weekend

What You'll Learn

  • The agent/environment loop: observation → action → reward
  • What PPO does at a high level
  • Why reward shaping is hard and easy to mess up
🧑‍🍳 Grown-Up Notes No API cost. Colab + Gymnasium + SB3 is self-contained. Lunar Lander and Breakout take a few GPU hours — not instant but feasible on free Colab.

Steps

  1. Install: 
    !pip install -q gymnasium[box2d] stable-baselines3[extra] moviepy
  2. Warm-up — CartPole (5 min run): 
    import gymnasium as gym
from stable_baselines3 import PPO
env = gym.make("CartPole-v1")
model = PPO("MlpPolicy", env, verbose=1)
model.learn(total_timesteps=50_000)
model.save("cartpole")
  3. Evaluate: 
    from stable_baselines3.common.evaluation import evaluate_policy
print(evaluate_policy(model, env, n_eval_episodes=10))
  4. Record a video of the trained agent playing.
  5. Main event — LunarLander (~30 min run): 
    env = gym.make("LunarLander-v3")
model = PPO("MlpPolicy", env, verbose=1, tensorboard_log="./tb/")
model.learn(total_timesteps=500_000)
  6. Launch TensorBoard in Colab: %load_ext tensorboard, then %tensorboard --logdir ./tb. Watch the reward curve go up.
  7. Record before/after videos. Write 3 observations about the agent's behavior.
  8. Stretch — Atari Breakout: use AtariWrapper and train for 2M steps. Requires GPU. Expect 2–4 hours.
🪄 If something goes wrong… Reward stuck at random? Try more timesteps — 50K isn't enough for LunarLander, aim for 500K. Agent learned a weird trick (e.g. flips upside down)? Reward hacking! Shape the reward to penalize it.

Try This Next

  • Try different algorithms. PPO vs DQN vs A2C on the same env. Which learns fastest?
  • Custom environment. Build your own Gym env for a game you designed.
  • Curriculum. Start easy, gradually crank difficulty. Faster learning?
