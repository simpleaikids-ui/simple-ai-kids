CNN From Scratch

Build and train a convolutional neural network on CIFAR-10 — then benchmark it against a pre-trained ResNet. Learn what convolutions actually do.

What You'll Make

A working CNN (~4 conv layers + 2 FC layers) trained on CIFAR-10 from scratch. You'll plot accuracy/loss curves, visualize learned filters, and compare to a fine-tuned ResNet-18 on the same data.

What You'll Need

  • Google Colab (T4 GPU) or local PyTorch setup
  • Comfort with Python + numpy basics
  • About a weekend

What You'll Learn

  • What a 2D convolution is, step by step
  • Why pooling and dropout help generalization
  • How transfer learning trounces from-scratch on small datasets
🧑‍🍳 Grown-Up Notes No API cost. Colab's free T4 handles this in ~20 min/epoch. Encourage the teen to print model and trace shapes through each layer — it's the best way to internalize CNNs.

Steps

  1. Open Colab, set runtime to GPU.
  2. Load CIFAR-10: 
    import torch, torchvision
from torchvision import transforms
tfm = transforms.Compose([transforms.ToTensor(), transforms.Normalize((0.5,)*3, (0.5,)*3)])
train = torchvision.datasets.CIFAR10(".", True,  tfm, download=True)
test  = torchvision.datasets.CIFAR10(".", False, tfm, download=True)
tl = torch.utils.data.DataLoader(train, 128, shuffle=True)
vl = torch.utils.data.DataLoader(test,  128)
  3. Define your CNN: 
    import torch.nn as nn, torch.nn.functional as F
class Net(nn.Module):
    def __init__(self):
        super().__init__()
        self.c1 = nn.Conv2d(3, 32, 3, padding=1)
        self.c2 = nn.Conv2d(32, 64, 3, padding=1)
        self.c3 = nn.Conv2d(64, 128, 3, padding=1)
        self.p  = nn.MaxPool2d(2)
        self.fc1 = nn.Linear(128*4*4, 256)
        self.fc2 = nn.Linear(256, 10)
        self.drop = nn.Dropout(0.3)
    def forward(self, x):
        x = self.p(F.relu(self.c1(x)))
        x = self.p(F.relu(self.c2(x)))
        x = self.p(F.relu(self.c3(x)))
        x = x.flatten(1)
        x = self.drop(F.relu(self.fc1(x)))
        return self.fc2(x)
  4. Train for 15 epochs. Track loss + test accuracy per epoch. Plot them.
  5. Target: reach ≥70% test accuracy on CIFAR-10.
  6. Visualize learned first-layer filters: 
    import matplotlib.pyplot as plt
w = model.c1.weight.data.cpu()
for i in range(8):
    plt.subplot(2,4,i+1); plt.imshow(w[i].permute(1,2,0).numpy()); plt.axis("off")
  7. Fine-tune ResNet-18 on the same data. You should hit 85%+ in a few epochs.
  8. Write a one-page reflection: why does transfer learning crush from-scratch on this size of data?
🪄 If something goes wrong… Accuracy plateaus at 30%? Your LR may be too high — try 1e-3 with Adam. Or too low — 1e-4 is sluggish. NaNs? Check normalization and use nn.CrossEntropyLoss (not MSE!) for classification.

Try This Next

  • Data augmentation. Add RandomHorizontalFlip, RandomCrop. Usually +5% accuracy.
  • Schedulers. Try CosineAnnealingLR. Faster convergence.
  • BatchNorm. Insert nn.BatchNorm2d after each conv. Often +3%.
