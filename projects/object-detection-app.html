Object Detection App

Use YOLO v8 to count cars in a parking lot, identify plants in your backyard, or read handwritten text in real time from your webcam.

What You'll Make

A small Python app that processes webcam or video input frame-by-frame and draws bounding boxes + labels on detected objects. Optionally save counts over time to a CSV.

What You'll Need

  • Python 3.10+ locally (or Colab for the first parts)
  • A webcam or video files to analyze
  • A weekend

What You'll Learn

  • How object detection is different from classification
  • The detection pipeline: anchors, NMS, IoU
  • How to pick a confidence threshold that balances precision and recall
🧑‍🍳 Grown-Up Notes Ultralytics YOLO is free for personal use (AGPL). Commercial use requires a license. No API cost. Pre-trained on COCO — recognizes 80 everyday objects out of the box.

Steps

  1. Install: 
    pip install ultralytics opencv-python
  2. Run detection on a sample image: 
    from ultralytics import YOLO
model = YOLO("yolov8n.pt")   # nano — small and fast
results = model("https://ultralytics.com/images/bus.jpg")
results[0].show()
  3. Point it at your webcam (Press Q to quit): 
    import cv2
cap = cv2.VideoCapture(0)
while True:
    ret, frame = cap.read()
    if not ret: break
    r = model(frame, verbose=False)[0]
    cv2.imshow("det", r.plot())
    if cv2.waitKey(1) == ord("q"): break
cap.release(); cv2.destroyAllWindows()
  4. Count class instances per frame. Log to CSV: 
    import csv, time
with open("counts.csv", "w", newline="") as f:
    w = csv.writer(f); w.writerow(["t","class","count"])
    # inside loop
    names = r.names
    cls_ids = r.boxes.cls.int().tolist()
    for c in set(cls_ids):
        w.writerow([time.time(), names[c], cls_ids.count(c)])
  5. Play with the confidence threshold (model(frame, conf=0.5)). Watch false positives drop as you raise it — but true positives disappear too.
  6. Pick one of three missions: count cars in a parking lot video, identify all plants in a garden clip, or detect handwritten digits (train on MNIST for this).
  7. Write a reflection: what types of objects does YOLO miss or confuse? Why?
🪄 If something goes wrong… Webcam shows black? Another app may have it — close Zoom, browser tabs. Bad detections? Try YOLOv8s or YOLOv8m (larger, slower, more accurate). Class not in COCO? Fine-tune on your own labeled dataset with model.train(data="your.yaml").

Try This Next

  • Train on your own classes. Use Roboflow to label 100 of your own images, export YOLO format, fine-tune.
  • Track objects across frames. Use model.track() so each car keeps the same ID.
  • Deploy as a web app. Wrap with Gradio or Streamlit, deploy on Hugging Face Spaces.
