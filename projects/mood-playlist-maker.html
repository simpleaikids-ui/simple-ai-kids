Mood Playlist Maker

Pull your music library's "audio features" from Spotify, cluster the songs by mood, and generate playlists automatically — happy, chill, angry, sad.

What You'll Make

A notebook that reads your library (or a public playlist), gets audio features (energy, valence, tempo…), runs K-means, and spits out 4 auto-generated playlists labeled by mood.

What You'll Need

  • Google Colab
  • A Spotify account + free developer app (for API keys)
  • Basic Python and a little patience
  • About 3 hours

What You'll Learn

  • What Spotify's "audio features" actually measure
  • How K-means clustering groups items without labels
  • Why unsupervised clusters rarely line up with human labels
🧑‍🍳 Grown-Up Notes Spotify developer account is free and requires an email signup. API keys go in a .env. Session-level tokens expire after 1 hour — normal. No purchases.

Steps

  1. Sign up at developer.spotify.com. Create an app. Copy your Client ID and Client Secret.
  2. In Colab: 
    !pip install -q spotipy scikit-learn matplotlib pandas
import spotipy
from spotipy.oauth2 import SpotifyClientCredentials
import os
os.environ["SPOTIPY_CLIENT_ID"]="…"
os.environ["SPOTIPY_CLIENT_SECRET"]="…"
sp = spotipy.Spotify(auth_manager=SpotifyClientCredentials())
  3. Pick a source playlist (any public one). Grab tracks: 
    PL = "37i9dQZF1DXcBWIGoYBM5M"   # Today's Top Hits
tracks = sp.playlist_tracks(PL)["items"]
ids = [t["track"]["id"] for t in tracks if t["track"]]
names = [t["track"]["name"] for t in tracks if t["track"]]
  4. Fetch audio features: 
    import pandas as pd
feats = sp.audio_features(ids)
df = pd.DataFrame(feats)[["danceability","energy","valence","tempo","acousticness"]]
df["name"] = names
  5. Cluster into 4 moods: 
    from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
X = StandardScaler().fit_transform(df[["danceability","energy","valence","tempo","acousticness"]])
df["cluster"] = KMeans(n_clusters=4, n_init=10, random_state=0).fit_predict(X)
  6. Label clusters by inspecting each group's averages. High valence + high energy = happy. Low energy + low valence = sad. Map them manually.
  7. Print your 4 playlists: 
    for c in sorted(df["cluster"].unique()):
    print(f"\n=== Cluster {c} ===")
    print(df[df["cluster"]==c]["name"].tolist()[:10])
  8. Sanity check: do the songs in "happy" actually feel happy? Write down 3 that don't, and explain why.
🪄 If something goes wrong… 401? Token expired — re-run the auth cell. Features = None? Spotify deprecated the audio-features endpoint for certain apps in 2024 — use a user token (OAuth flow) instead of client-credentials.

Try This Next

  • Visualize in 2D. Use PCA or UMAP to plot clusters. Color by K's guess.
  • K tuning. Try K=3, 5, 6. Which feels most like real moods?
  • Hybrid ranking. Blend clusters with play count or release year.
