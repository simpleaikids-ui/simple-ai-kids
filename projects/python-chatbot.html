Python Chatbot

Build a chatbot in pure Python — first as a rule-based toy, then upgrade it to a real AI chatbot using an API.

What You'll Make

Two versions of the same chatbot: Version 1 uses if/elif rules (~50 lines). Version 2 calls Claude or GPT via an API (~30 lines). You'll compare how they behave.

What You'll Need

  • A Google account (for Colab)
  • Basic Python (functions, dicts, if/elif)
  • Anthropic or OpenAI API key — parent required if under 18
  • About 2–3 hours

What You'll Learn

  • How simple rule-based bots work and where they break
  • How to call an LLM API from Python
  • How system prompts shape an AI's behavior
🧑‍🍳 Grown-Up Notes The API portion costs ~\$0.10. If your teen is under 18, create the API account with your email and share the key via a .env file. Discuss rate limits and the importance of never committing keys to GitHub.

Steps

  1. Open colab.research.google.com, start a new notebook.
  2. Version 1 — Rule-based: 
    def bot(msg):
    msg = msg.lower()
    if any(w in msg for w in ["hi", "hello", "hey"]):
        return "Hey there!"
    if "bye" in msg:
        return "See ya 👋"
    if "name" in msg:
        return "I'm a Python bot!"
    if "?" in msg:
        return "Hmm, good question. I'm just a baby bot."
    return "I don't know what that means."

while True:
    m = input("You: ")
    if not m: break
    print("Bot:", bot(m))
  3. Chat with Version 1 for 2 minutes. Write down 3 questions it can't handle.
  4. Version 2 — API-powered: 
    !pip install anthropic
import os
from anthropic import Anthropic
os.environ["ANTHROPIC_API_KEY"] = "sk-ant-…"   # paste here
client = Anthropic()

def bot(history):
    msg = client.messages.create(
        model="claude-haiku-4-5-20251001",
        max_tokens=300,
        system="You are a friendly chatbot for teens. Keep replies short and fun.",
        messages=history,
    )
    return msg.content[0].text

history = []
while True:
    m = input("You: ")
    if not m: break
    history.append({"role": "user", "content": m})
    reply = bot(history)
    history.append({"role": "assistant", "content": reply})
    print("Bot:", reply)
  5. Chat with Version 2. Ask it the 3 questions Version 1 failed on.
  6. Change the system prompt — make it a grumpy pirate. Notice how the tone changes.
  7. Track how much memory (history) the bot has. What happens after 20 turns?
🪄 If something goes wrong… 401 Unauthorized = key is wrong. 429 = you're rate-limited, wait a minute. ANTHROPIC_API_KEY missing = make sure you pasted it into the os.environ line.

Try This Next

  • Add streaming. Use client.messages.stream so replies type out word-by-word.
  • Add personas. Let the user pick between "Pirate", "Robot", "Shakespeare." Each has a different system prompt.
  • Add file input. Let the bot read a text file and answer questions about it (mini-RAG!).
