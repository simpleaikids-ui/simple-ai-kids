Dance Move Detector

Teach the computer your best dance poses — then play a pose-matching game where the screen shows a move and you have to nail it!

What You'll Make

A full-body pose game. You train the model on 4 dance moves (like T-pose, flamingo, ninja, robot), then a Scratch game flashes a move on screen and scores you for matching it fast.

What You'll Need

  • A computer with a webcam and enough space to stand back from it
  • A free Machine Learning for Kids account
  • About 60 minutes
  • Room to dance!

What You'll Learn

  • How body pose detection is different from object classification
  • Why consistent lighting and a clean background help
  • How to score players on reaction time in Scratch
🧑‍🍳 Grown-Up Notes The child needs space to move — 2 meters back from the webcam is ideal. Warn younger siblings not to walk into the frame during training (it confuses the model). Same platform as Rock-Paper-Scissors (ML for Kids).

Steps

  1. In Machine Learning for Kids, create a new Image project called DanceMoves.
  2. Add four classes: tpose, flamingo (one leg up), ninja (arms crossed), robot (arms in stiff angles).
  3. Stand back from the webcam. For each move, take 40 photos — face the camera, slight left, slight right, arms slightly different each time.
  4. Add a nothing class with 30 photos of the empty room.
  5. Train the model. Test each pose — should get ~90% confidence on the right class.
  6. Open Scratch via ML4K's Make → Scratch 3.
  7. Create a new project. Add 4 costume sprites, one per move (you can draw stick figures).
  8. Build the game loop: 
    when green flag clicked
set [score v] to 0
repeat (10)
  pick random costume (1 to 4)
  say [Do THIS pose!] for 1 sec
  wait (3) secs
  if <(recognise current pose) = (current costume name)> then
    change [score v] by 1
    say [✓] for 0.5 secs
say (join [Final score: ] (score))
  9. Play! Try to get 10/10.
🪄 If something goes wrong… Model is 50/50 between two poses? They look too similar to the webcam. Exaggerate — bigger arm angles, wider stance. Also retrain with 20 more examples.

Try This Next

  • Add your own moves. Invent 4 new poses. Name them whatever you want.
  • Add a timer. Only count the score if the pose matches within 2 seconds.
  • Multiplayer. Two players take turns. High score wins.
