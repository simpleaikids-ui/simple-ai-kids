Sentiment Analyzer

Score tweets, lyrics, or game reviews as positive or negative — then chart the mood of an artist, a game, or a season of your favorite show.

What You'll Make

A Python program that takes a bunch of text (reviews, lyrics, tweets) and tags each one positive/negative/neutral. Visualize the results as a bar chart or time-series line.

What You'll Need

  • Google Colab
  • A text dataset (Kaggle reviews, lyric API, exported tweets)
  • About 90 minutes

What You'll Learn

  • How pre-trained sentiment models work (zero-shot, no training needed)
  • How to wrangle messy real-world text
  • When sentiment models fail (sarcasm, slang, code-switching)
🧑‍🍳 Grown-Up Notes Use public datasets — Kaggle's "Amazon Reviews" or a lyric API. Scraping Twitter/X requires paid API access. Good moment to talk about the limits of sentiment analysis (false positives on "this is sick!").

Steps

  1. Pick your corpus. Easy start: Kaggle Amazon Reviews (small). Medium: lyrics from a single artist. Hard: exported Discord messages.
  2. In Colab, install: 
    !pip install -q transformers pandas matplotlib
  3. Load a small pretrained sentiment model: 
    from transformers import pipeline
clf = pipeline("sentiment-analysis")   # default is distilbert-base-uncased-finetuned-sst-2-english
print(clf("This album is incredible"))
print(clf("Worst experience of my life"))
  4. Load your data: 
    import pandas as pd
df = pd.read_csv("reviews.csv").head(200)
  5. Score every row. Batch it so it's fast: 
    texts = df["text"].astype(str).tolist()
results = clf(texts, truncation=True)
df["sentiment"] = [r["label"] for r in results]
df["confidence"] = [r["score"] for r in results]
  6. Chart it: 
    import matplotlib.pyplot as plt
df["sentiment"].value_counts().plot.bar()
plt.title("Sentiment distribution"); plt.show()
  7. Find the bot's mistakes. Sort by confidence ascending — low-confidence predictions are often errors. Read 10. What patterns do you see?
  8. Honesty round: manually label 30 rows. Compare to the model. What's its accuracy? Which types of text fail?
🪄 If something goes wrong… CUDA out of memory? Use clf(texts, batch_size=8). Sequence too long? truncation=True. Getting all POSITIVE? Check — if your data really is all positive, that's correct, not a bug.

Try This Next

  • Time series mood. Chart sentiment per month. Is the artist getting happier?
  • Multilingual. Swap to nlptown/bert-base-multilingual-uncased-sentiment. Analyze Spanish or French text.
  • Sarcasm hunt. Train a tiny classifier on a sarcasm dataset and compare.
